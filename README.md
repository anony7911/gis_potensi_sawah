# gis_potensi_sawah

# install
  - git clone https://github.com/anony7911/gis_potensi_sawah.git
  - cd gis_potensi_sawah
  - composer update --no-scripts
  - npm install
  - npm run dev
  - open new terminal and run 'php artisan serve'
